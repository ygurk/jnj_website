Default surface container for grouping content — features, testimonials, pricing, etc.

```jsx
<Card>
  <h4>Fast, friendly websites</h4>
  <p>Copy goes here.</p>
</Card>

<Card tone="brand" padding="lg">Dark pine panel</Card>
<Card interactive>Clickable card with hover lift</Card>
```

Tones: `white | cream | brand | soft`. Padding: `none | sm | md | lg`. Set `interactive` for clickable cards, `as="a"` to make the whole card a link.
